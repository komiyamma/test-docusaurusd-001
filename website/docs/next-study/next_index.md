## 図解①：App Router の「フォルダ＝画面」感 🗂️🧭

```mermaid
flowchart TD
  A[app/] --> B[layout.tsx<br/>全ページ共通の外枠]
  A --> C[page.tsx<br/>そのURLの画面]
  A --> D[loading.tsx<br/>読み込み中UI]
  A --> E[error.tsx<br/>エラーUI]
  A --> F[not-found.tsx<br/>404 UI]
  A --> G["blog/"]
  G --> H[page.tsx<br/>/blog]
  G --> I["[slug]/"]
  I --> J[page.tsx<br/>/blog/:slug]
```

## 図解②：リクエストが来た時の流れ（ざっくり）📨➡️🧠

```mermaid
flowchart LR
  U[Browser] --> R[Next.js Router]
  R --> M{Middleware?}
  M -->|Yes| MW[Middleware処理]
  M -->|No| P[Routeへ]
  MW --> P
  P --> SC[Server Componentsで描画]
  SC --> ST[Streaming / Suspense]
  ST --> U
  SC --> CC[Client Components必要ならJS配信]
  CC --> U
```
